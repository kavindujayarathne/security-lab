# Security Lab

Security Lab is my personal virtual homelab, a controlled environment where I run attack simulation, defensive monitoring, detection engineering, malware research, incident response, and security-focused experimentation.

The lab is designed as a long-term project. It grows over time as new techniques, tools, scenarios, and workflows are added.

## Mission Statement

To understand how attacks unfold, how defenses respond, and what each side leaves behind, by running every scenario end to end inside a controlled environment.
